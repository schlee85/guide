Button

버튼 컴포넌트.

기본

button/a/span 태그에 .btn 부여.



a tag span tag 

<button type="button" class="btn"><span>default</span></button>
<button type="button" class="btn primary"><span>primary</span></button>
<button type="button" class="btn secondary"><span>secondary</span></button>

<a href="#" class="btn"><span>a tag</span></a>
<span class="btn"><span>span tag</span></span>

비활성

[disabled] 속성 부여.


<button type="button" class="btn" disabled><span>default</span></button>
<button type="button" class="btn primary" disabled><span>primary</span></button>
<button type="button" class="btn secondary" disabled><span>secondary</span></button>

외곽선

.outline 추가.


<button type="button" class="btn outline"><span>default</span></button>
<button type="button" class="btn primary outline"><span>primary</span></button>
<button type="button" class="btn secondary outline"><span>secondary</span></button>

크기

.small / .large 추가


<button type="button" class="btn small"><span>small</span></button>
<button type="button" class="btn"><span>default</span></button>
<button type="button" class="btn large"><span>large</span></button>

아이콘

아이콘 컴포넌트 삽입.


<button type="button" class="btn">
	<i class="icon"></i><span>default</span>
</button>
<button type="button" class="btn">
	<span>default</span><i class="icon"></i>
</button>
<button type="button" class="btn"><i class="icon"></i></button>

그룹

.btn_wrap 으로 감쌈.


<div class="btn_wrap">
	<button type="button" class="btn"><span>default</span></button>
	<button type="button" class="btn"><span>default</span></button>
	<button type="button" class="btn"><span>default</span></button>
</div>
